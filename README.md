# Epi-Spatial-Temporal-Teilen

Accurate epidemic forecasting is crucial for effective disease control and prevention. Traditional compartmental models often struggle to estimate epidemiological parameters that vary temporally and spatially, while deep learning models typically overlook underlying disease transmission dynamics and lack interpretability grounded in epidemiological theory. To address these challenges, we propose the Causal Spatiotemporal Graph Neural Network (CSTGNN)—a novel hybrid framework that integrates a Spatio-contact SIR model with the Graph Neural Networks (GNNs) to capture the propagation of epidemics over time and across regions. Inter-regional human mobility exhibits continuous and smooth spatiotemporal dynamics, resulting in adjacent graph structures sharing underlying mobility patterns. To model these shared mobility patterns, we employ an adaptive static connectivity graph to represent the stable components of human mobility and utilize a temporal dynamics model to learn the fluctuations within these mobility patterns. By integrating the adaptive static connectivity graph with the temporal dynamics graph, we construct a dynamic graph that encapsulates the comprehensive properties of human mobility networks. To capture the temporal trends and variations in infectious disease spread, we introduce a temporal decomposition model to handle temporal dependencies. This model is then integrated with a dynamic graph convolutional network for epidemic forecasting. We validate our model using real-world datasets at the provincial level in China and the state level in Germany. Extensive experiments demonstrate that our method accurately simulates the spatiotemporal dynamics of infectious diseases, providing a valuable tool for epidemic forecasting and intervention strategies. Furthermore, by analyzing the learned parameters, we gain deeper insights into the mechanisms of disease transmission, thereby enhancing the interpretability of our model.
